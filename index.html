Welcome to the DISSECT-CF documentation website

This webpage contains the most recent API guides for DISSECT-CF, the DIScrete event baSed Energy Consumption simulaTor for Clouds and Federations. For other details, source code and up-to-date information about the simulator, please visit its main github page.

Support or Contact

Apart from the documentation listed here, one can find plenty of useful information about the way the simulator is intended to work:

  • First of all, more elaborate examples can be found in the dissect-cf-examples project.
  • Also, the wiki provides further insights to advanced topics (like creating custom schedulers) and offers a FAQ. Apart from contributing with code, feel free to contribute there with documentation as well.
  • Do you still have some questions? If so, then please share them with the simulator's user and developer community at our Q&A forum - a registration is needed to send your questions in.
